# gitee.com/Ljolan/si-mqtt

#### 介绍
golang mqtt服务器，集群版【设计中】

#### 使用说明

- core 包为核心包
- 项目基础配置在core/config/config.toml里面
- 添加环境变量 SI_CFG_PATH = "配置文件路径" ，如果不配置，则默认使用core/config/config.toml配置
- 以package方式 运行 main.go即可
